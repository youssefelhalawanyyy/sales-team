## PART 1: HOW TO DOWNLOAD & INSTALL (For Your Users)

### FOR macOS USERS (Apple Computer):

#### **Step 1: Download the App**
1. Go to your download link (your website or Google Drive)
2. Look for file named: `Sales Team-0.1.0-arm64.dmg`
3. Click to download (it's ~161 MB, will take 1-2 minutes)
4. Wait for it to finish downloading
5. Look in your Downloads folder

#### **Step 2: Install the App**
1. Find the downloaded file: `Sales Team-0.1.0-arm64.dmg`
2. Double-click on it
3. A window appears showing "Sales Team" app icon
4. Drag the "Sales Team" icon to the "Applications" folder
5. Wait for copy to finish (30 seconds - 1 minute)
6. Close the window
7. You can delete the `.dmg` file now (not needed anymore)

#### **Step 3: Launch the App**
1. Open Applications folder (or use Spotlight search)
2. Search for "Sales Team"
3. Double-click to launch
4. First time might take 5-10 seconds to start
5. Done! App is running

#### **Step 4: Updates (After We Add Auto-Update)**
1. App checks for updates automatically
2. You see notification: "New version available"
3. Click "Update" button
4. App downloads update automatically (in background)
5. When done: "Restart to apply update?"
6. Click "Restart Now" or choose time later
7. App closes, updates, and reopens automatically
8. **NO REINSTALL NEEDED!**

### FOR WINDOWS USERS (PC):

#### **Step 1: Download the App**
1. Go to your download link (your website or Google Drive)
2. Look for file named: `Sales Team Setup 0.1.0.exe`
3. Click to download (it's ~150 MB, will take 1-2 minutes)
4. Wait for it to finish downloading
5. Look in your Downloads folder

#### **Step 2: Install the App**
1. Find the downloaded file: `Sales Team Setup 0.1.0.exe`
2. Double-click on it
3. Click "Yes" if Windows asks "Allow this app to make changes?"
4. Wait for installer to launch (might take 10-15 seconds)
5. Follow the steps in the installer:
   - Click "Next"
   - Click "I Agree" (license)
   - Click "Next" (location)
   - Click "Install"
6. Wait for installation (1-2 minutes)
7. Click "Finish" - app launches automatically!

#### **Step 3: Launch the App**
1. Double-click "Sales Team" icon on Desktop
2. OR search Start menu for "Sales Team"
3. App opens (first time takes 5-10 seconds)
4. Done! App is running

#### **Step 4: Updates (After We Add Auto-Update)**
1. App checks for updates automatically
2. You see notification: "New version available"

4. App downloads update automatically (in background)

## PART 2: WHAT TO UPLOAD FOR PEOPLE TO DOWNLOAD

### **Option A: Google Drive (EASIEST FOR BEGINNERS)**

#### Step 1: Create a Google Drive folder
1. Go to https://drive.google.com
2. Click "New" → "Folder"
3. Name it: "Sales Team App Downloads"
4. Right-click folder → "Share"

#### Step 2: Share the folder
1. Click "Change to anyone with the link"
2. Make sure "Viewer" is selected (they can only download)
3. Click "Copy link"
4. Send this link to users

#### Step 3: Upload installer files
1. Click into the folder you created
2. Click "File upload" or drag files into folder
3. For macOS users: Upload **`Sales Team-0.1.0-arm64.dmg`**
4. For Windows users: Upload **`Sales Team Setup 0.1.0.exe`**
5. Optional: Upload **`Sales Team-0.1.0-arm64-mac.zip`** as backup

1. Create a new file in Google Drive (click "New" → "Document")
2. Name it: "Installation Instructions"

4. Done! Find it in Applications folder
